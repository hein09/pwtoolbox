To use Vipster as a Python library, import it via from vipster import *.

This provides the following objects:

Molecule 

class Molecule(name,steps)

Main data container for periodic molecular data.

In-detail explanation here.

readFile 

function readFile(fn,fmt,mode) -> Molecule,Parameter

Reads fn as file of type fmt and returns an instance of vipster.Molecule and either a dictionary containing reconstructed calculation parameters or None.

Supported file-formats are given in inFormats.

mode is either "cli" or "gui". This decides which format-name-convention is used (short-hand, see inFormats or long, see GUI-selector). The default is "cli" and can therefore be ignored.

writeFile 

function writeFile(mol,fmt,fn,param,coordfmt,mode)

Writes mol and param (default: None) to file fn.

fmt is the file-format, supported file-formats are given in outFormats.

coordfmt is the format in which the coordinates are saved. The default is "bohr", if the file-format does not support multiple coordinate formats the argument is ignored.

mode is either "cli" or "gui". This decides which format-name-convention is used (short-hand, see outFormats or long, see GUI-selector). The default is "cli" and can therefore be ignored.

inFormats 

list inFormats

Contains the supported short-hand strings for the file-format arguments in readFile.

outFormats 

list outFormats

Contains the supported short-hand strings for the file-format arguments in writeFile.

config 

dict config

Contains various global settings, e.g. bond-cutoff-factor and GUI-settings.

pse 

dict pse

Global periodic system of the elements.

Every instance of Molecule has a local copy and fetches missing entries from here.

default 

dict default

The source dictionary for config and pse.

Acts as fallback for restoring changed data in child-dictionaries. Will be parsed either from file default.json or ~/.vipster.json, if present.

saveConfig 

function saveConfig()

Saves config and pse to ~/.vipster.json.

newParam 

function newParam(prog,preset) -> Param

Returns a new and empty parameter-dictionary of type prog.

Currently, only "pwi" and "lmp" supported.

preset chooses pre-filled presets, of which currently none are implemented.

launchVipster 

function launchVipster(m,p)

Launches the GUI.

m and p are the lists of loaded Molecules and parameter files and can be supplied explicitely. If not supplied, fallback arguments GuiMolecules and GuiParameters are used instead.

GuiMolecules 

list GuiMolecules

Default list for Molecules if GUI is started without explicit arguments.

GuiParameters 

list GuiParameters

Default list for parameter files if GUI is started without explicit arguments.
