Vipster offers help in both editing and visualizing chemical data. A standalone executable script is provided, which directly launches the graphical interface, where data can be pre- and post-processed.

Additionally, it can be loaded from within the Python-interpreter or a script via from vipster import *, which exposes among other the Molecule class, which is the main data container, and the readFile and launchVipster functions, which offer I/O- and GUI-capabilities.

Various file types are supported:

  • standard xyz
  • Empire-xyz
  • PWScf input/output
  • Lammps data/custom-dump
  • Gaussian cube
  • AIMALL output
