Vipster is a powerful and versatile tool for pre- and post-processing of periodic chemical structures.

It has a beatiful graphical interface with a few rather rare and a few rather useless functions. Something for everyone, i hope!

And if you're missing something, it has a simple python interface, allowing you to enhance it. Or your scripts. Or don't touch it, your call!

It can read the following files:

  • standard xyz
  • Empire-xyz
  • PWScf input/output
  • LAMMPS data/custom-dump
  • Gaussian cube files
  • AIMALL output
  • CPMD input
  • Mol2
  • (A)XSF
  • Turbomole

And write the following:

  • xyz
  • Empire-xyz
  • PWScf input
  • LAMMPS data
  • CPMD input
